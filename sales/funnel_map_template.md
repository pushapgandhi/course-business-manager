| Stage | Asset Or Tool | Message Goal | CTA | Owner |
|---|---|---|---|---|
| Awareness |  | Help the audience recognize the problem |  | Marketing Agent |
| Lead capture |  | Collect interest or contact details |  | Sales Agent |
| Trust building |  | Explain value and reduce hesitation |  | Marketing + Communication |
| Conversion |  | Invite the buyer to join, inquire, book, or pay |  | Sales Agent |
