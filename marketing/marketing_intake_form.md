# Marketing Content Intake Form

Use this form when the user asks for marketing content, Instagram posts, content calendars, repurposing, or social media growth assets.

## Required Details

| Field | User Input |
|---|---|
| Audience |  |
| Content goal |  |

| Source asset or topic |  |

## Optional Details

| Field | User Input |
|---|---|
| Format | Carousel, reel, story, single post, calendar, newsletter, blog, or repurposing table |
| Course, program, or offer |  |
| CTA | Save, comment, follow, DM, link in bio, join waitlist, or download resource |
| Brand voice | Clear, practical, beginner-friendly, professional |
| Source approval status | Approved, draft, user-provided, or assumption-based |
| Posting cadence | Daily, 3x/week, weekly, launch period, or custom |
| Design stack | Canva Free, Canva Pro, designer, AI image/design, or manual |
| Publishing stack | Meta Business Suite, manual posting, or external scheduler |
| Hashtag preference | Niche, broad, local, branded, or none |
| Restrictions | Claims, words to avoid, topics to avoid, compliance notes |

## Minimal Intake Questions

Ask only what is needed:

1. Who is this content for?
2. What should the content accomplish?
3. What source asset, topic, or offer should I use?
4. Which Instagram format do you want, or should I choose?
5. What CTA should the post use?

## Default Choices

When the user does not specify:

- Platform: Instagram
- Goal: audience growth
- Format: choose the best fit for the topic
- CTA: save, comment, or follow for awareness content
- Design stack: Canva Free or manual design
- Publishing stack: Meta Business Suite or manual posting
- Approval status: draft until user approves
