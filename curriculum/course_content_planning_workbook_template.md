# Course Content Planning Workbook Template

## Purpose

Use this template to create the Excel `.xlsx` workbook that visually shows what course content will be taught, module by module and lesson by lesson.

The workbook is the approved bridge between Curriculum & Course Planning and Lesson Content & Teaching Scripts. The Lesson Content agent uses approved workbook rows as source truth for presentations, video scripts, and teaching notes.

## Workbook File

Recommended file name:

```text
[course-name]-course-content-planning-workbook.xlsx
```

## Required Tabs

| Tab | Purpose |
|---|---|
| Course Overview | Summarize the course promise, learner, outcome, format, and review status |
| Module Map | Show the module sequence and the purpose of each module |
| Lesson Map | Show each lesson, objective, key concepts, practice, and student output |
| Assignments And Checkpoints | Track practice tasks, assignments, checkpoints, and capstone milestones |
| Lesson Content Handoff | Track what is approved and ready for presentation, video script, and teaching-note production |

## Tab 1: Course Overview

| Column | Required | Notes |
|---|---|---|
| Course title | Yes | Working or approved course name |
| Target learner | Yes | Audience, role, skill level, and context |
| Starting state | Yes | What students know or have before the course |
| Final outcome | Yes | What students can do by the end |
| Course promise | Yes | Clear transformation statement |
| Course format | Yes | Live, recorded, cohort-based, self-paced, hybrid, workshop, or bootcamp |
| Duration | Yes | Weeks, sessions, lessons, or total hours |
| Skill level | Yes | Beginner, intermediate, advanced, or mixed |
| Delivery platform | No | LMS, Google Drive, Zoom, community, Notion, etc. |
| Curriculum review status | Yes | Draft, Needs revision, or Approved |
| Approved by | No | User or reviewer name |
| Approval date | No | Date approved for lesson production |
| Notes | No | Constraints, risks, or special instructions |

## Workbook Review Rules

- Lesson Content can start only when the relevant workbook row, lesson, module, or batch is marked Approved.
- If a needed module, lesson, objective, outcome, or assignment is missing, return the workbook to Curriculum for revision.
- Do not add new modules, lessons, or outcomes inside Lesson Content without user approval.
- Keep workbook statuses current after each production batch.

## Visual Formatting Recommendations

- Freeze the header row on every tab.
- Use filters on every table.
- Use one row per module, lesson, assignment, or handoff item.
- Use color-coded status cells: Draft, Needs revision, Approved, Produced.
- Keep wording short enough for scanning.
