# Scheduling & Calendar Templates

Use these templates for calendar details, invites, reschedules, cancellations, reminders, and prep.

## Calendar Event Details

```text
Title:
Attendees:
Date:
Time:
Timezone:
Duration:
Location/link:
Description:
Reminders:
```

## Meeting Invite Description

```text
Purpose:
[Short reason for the meeting]

Agenda:
1. [Topic 1]
2. [Topic 2]
3. [Decision or next step]

Preparation:
- [What attendees should review or bring]
```

## Recipient Meeting Email

```text
Subject: [Meeting title] - Meeting Details

Hi [Recipient name],

Confirming our meeting details:

Meeting: [Meeting title]
Date/time: [Day, Month Date, Time, Timezone]
Location/link: [Meeting link or location]

Agenda:
1. [Topic 1]
2. [Topic 2]
3. [Decision or next step]

Best,
[Sender name]
```

## Proposed Time Options

```text
I found these possible options:

1. [Day, Month Date, Time, Timezone]
2. [Day, Month Date, Time, Timezone]
3. [Day, Month Date, Time, Timezone]

Please approve one option or suggest another time range.
```

## Reschedule Plan

```text
Original meeting:
- Title:
- Current date/time:
- Attendees:

Proposed new time:
- Date/time:
- Timezone:
- Duration:
- Conflict status:

Notification needed:
- Yes/No
```

## Cancellation Plan

```text
Meeting to cancel:
- Title:
- Date/time:
- Attendees:

Reason to include, if any:
[Reason]

Notification needed:
- Yes/No
```
